# Example app for CI Hexlet course

[![workflow.yml][image-1]][1]
Starting boilerplate of [Strapi][2] application

## System requirements

* NodeJS \>= 18
* NPM \>= 6.x
* Make

## Using

```sh
make setup
make start
```

## Run tests

```sh
make test
```

## Run linter

```sh
make lint
```
